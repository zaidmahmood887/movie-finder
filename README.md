# Movie Finder

## Projectbeschrijving
Movie Finder is een interactieve single-page webapplicatie gebouwd met Vite en JavaScript.

Gebruikers kunnen films zoeken, filteren, sorteren en opslaan als favorieten.

## Functionaliteiten

- Zoekfunctie
- Sorteren op naam en rating
- Genre filter
- Favorieten opslaan
- Dark mode
- Responsive design

## Gebruikte API

TVMaze API:
https://api.tvmaze.com/shows
